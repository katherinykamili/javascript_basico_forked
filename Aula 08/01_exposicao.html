O que são Funções?

Funções são blocos de código reutilizáveis. Assim como uma máquina que recebe entrada e produz saída, uma função recebe dados (parâmetros), processa e retorna um resultado.

function somar(a, b) {
return a + b;
} console.log(somar(3, 5)); // 8

Declaração de função

Declaramos funções com a palavra-chave function. Ela pode ser chamada pelo nome quantas vezes quisermos. Isso evita repetir código.

function saudacao() {
console.log("Olá!"); }
saudacao();

Parâmetros e argumentos

Parâmetros são variáveis que a função recebe. Argumentos são os valores passados ao chamar a função.

function dobro(x) {
return x * 2;
}
console.log(dobro(5)); // 10

Retorno com return

return devolve um valor da função. Sem return, a função retorna undefined. O return também encerra a execução da função.

function quadrado(n) {
return n * n;
}
let resultado = quadrado(4); // 16

Escopo em funções

Variáveis declaradas dentro de funções (com let/const) só existem lá dentro - escopo local. Variáveis fora são globais.

let global = "OK"; function teste() {
let local = "interno";
console.log(global); // OK }

Arrow functions

Arrow functions são uma forma mais curta de escrever funções: () => { }. Se só tem uma linha, pode omitir {} e return.

let dobro = (x) => x * 2;
console.log(dobro(5)); // 10

Funções como argumento (callbacks)

Callbacks são funções passadas como argumento para outra função. São muito usadas em arrays (filter, map, forEach).

let nums = [1, 2, 3];
nums.forEach(function(n) {
console.log(n); }
);

Parâmetros rest e default

Parâmetro default define valor padrão caso nenhum argumento seja passado. Parâmetro rest (...args) agrupa múltiplos argumentos em um array.

function somar(...nums) {
return nums.reduce((a, b) => a + b);
} console.log(somar(1, 2, 3)); // 6
